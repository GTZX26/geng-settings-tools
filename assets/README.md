# Assets (no binary in git)

โปรเจกต์นี้หลีกเลี่ยงการ commit ไฟล์ binary ลงใน repo โดยตรง
(เช่น PNG/JPG) เพื่อให้ผ่านข้อกำหนดรีวิวที่ "ไม่รองรับไบนารี".

ถ้าต้องการใช้โลโก้จริงของแอป:

1. เตรียมไฟล์ `icon.png` ขนาดอย่างน้อย 512x512
2. เตรียมไฟล์ `qrcode.png` (ถ้ามี)
3. วางไฟล์ทั้งสองที่ `/usr/share/geng-setting-tools/`

แอปจะทำงานได้แม้ไม่มีไฟล์รูป (จะซ่อนรูปอัตโนมัติ).
